# Maine Office of GIS (MEGIS) — Legacy Catalog

**System:** Legacy catalog interface (being superseded by Maine GeoLibrary Hub)
**URL:** https://www.maine.gov/megis/catalog/
**Publisher:** State of Maine — Office of GIS (MEGIS)

## Status

- **Legacy / deprecated.** The site displays a prominent notice: *"THE MAINE GEOLIBRARY WEBSITE HAS BEEN UPGRADED TO A NEW PLATFORM. THIS SITE IS NOW OBSOLETE AND WILL BE DISCONTINUED IN THE FUTURE."*
- **Replacement:** https://mainegeolibrary-maine.hub.arcgis.com/

## What it covers (for historical context)

The legacy catalog provided metadata records for statewide GIS layers including:

| Layer | Relevance to South Portland setbacks |
|-------|--------------------------------------|
| **Parcels** | Tax parcel boundaries (organized towns) — same data as what South Portland publishes directly on its own Hub |
| **Elevation** | 10-meter DEMs and LiDAR-derived terrain — useful for slope analysis but not setback rules directly |
| **Hydrography** | Streams, lakes, wetlands — relevant for shoreland setback measurement ( setbacks measured from high-water mark) |
| **Transportation** | Roads, rails, trails — used as the street centerline reference for front-yard classification |
| **Conservation Lands** | Public and private conservation easements — may affect buildable area |

## Recommended action

Use the **new Maine GeoLibrary Hub** instead: https://mainegeolibrary-maine.hub.arcgis.com/
